## Rodando

* Necessário [Node.js](https://nodejs.org/) e NPM instalados.

Clone o projeto, entre na pasta e instale as dependências:
```
npm install
```

E rode o projeto:
```
ng serve
```

Acesse em [localhost:4200](http://localhost:4200/)
